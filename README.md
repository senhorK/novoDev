# 🎮 Jogos em Desenvolvimento

## Geometry Style Game
Um jogo inspirado no estilo Geometry Dash, com:

- sistema de pulo e duplo pulo
- obstáculos dinâmicos
- partículas visuais
- efeitos neon
- progressão por fases
- menu de seleção de níveis
- otimização de renderização

Projeto focado em lógica de gameplay e performance.
